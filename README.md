# Chatboto

This is a simple desktop app built with [Iced](https://iced.rs) that allows you to chat with AI:
 - **Mistral**
 - **Gemini**

## Installation

1. Clone this repository:
```bash
git clone https://github.com/j03-dev/chatboto.git
```

2. Clone this repository:
```bash
cargo build
````
